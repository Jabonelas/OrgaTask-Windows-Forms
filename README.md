## 📋 Visão Geral
Aplicação desktop que se conecta à OrgaTask API para:
- Autenticação de usuários
- Visualização de tarefas
- Atualização de status
- Gerenciamento de prioridades

## 🌐 Ecossistema OrgaTask
- **Backend**: [OrgaTask API](https://github.com/Jabonelas/OrgaTask-API)
- **Versão Web**: [OrgaTask Blazor](https://github.com/Jabonelas/OrgaTask-Blazor-WebAssembly)

## 🛠 Tecnologias
- Windows Forms
- .NET Framework 4.8
- Injeção de Dependência
- Design Patterns (Repository, Service Layer)
- HttpClient para consumo de API
